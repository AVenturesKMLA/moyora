# 🌐 모여라 (Moyeora) 프로젝트 요약

## 📌 한 줄 소개
> **전국 고등학교 동아리들이 모여 협력하고 성장하는 플랫폼**

---

## 🎯 주요 기능

| 기능 | 설명 |
|------|------|
| 🏫 **동아리 등록** | 학교별 동아리를 등록하고 관리 |
| 📅 **이벤트 관리** | 대회, 포럼, 공동연구 등 이벤트 생성/참여 |
| 💬 **실시간 채팅** | 동아리 간 소통 및 협업 |
| 🔔 **알림 시스템** | 새 이벤트, 신청 현황 등 실시간 알림 |
| 👤 **마이페이지** | 내 활동 내역, 참여 이벤트 확인 |
| 🌙 **다크/라이트 모드** | 사용자 취향에 맞는 테마 선택 |

---

## 💻 기술 스택 (비개발자용)

### 프론트엔드 (사용자가 보는 화면)
- **Next.js** - React 기반 웹 프레임워크
- **TypeScript** - 안정적인 코드를 위한 도구
- **TailwindCSS** - 깔끔한 디자인 시스템

### 백엔드 (서버에서 처리)
- **MongoDB** - 데이터베이스 (사용자, 동아리, 이벤트 정보 저장)
- **NextAuth** - 로그인/회원가입 시스템

### 특별 기능
- **3D 네트워크 글로브** - 메인 페이지의 인터랙티브 지구본 애니메이션
